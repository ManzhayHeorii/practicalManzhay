Вхід Реєстрація

Create a Repository

From scratch -- Create a new local repository

$git init [name]

Download from an existing repository

$git clone my_url

Observe your Repository

List new modified files not yet commited

$git status

Show the changes to files not yet stugged

$git diff

Show all changes to stagged file

$git diff -- catched

Show all stagged and unstagged changes

$git diff HEAD

Show changes between two commits ;ids

$git diff commit1 commit2

List the change dates authors for a file

$git blame[file]

Show the changes for a commit id

$git show[commit]:[file]

Show full history

$git log

Working with Branches

List local branches

$git branches

List all branches

$git branch -av

Switch branch

$git checkout my_branch

Create new branch

$git branch new_branch

Delete branch

$git branch -d ny_branch

Merge branch

$git checkout branch branch_b

$git checkout merge branch_a

Make a change

Stages the file ready to commit

$git add [file]

Stage all changes file ready to commit

$git add .

Commit all changed files

$git commit -m "commit message"

Synchronize

Get the lattest changes from origin

$git fetch

Fetch the lastest changes from origin

$git pull

Fetch the lastest changes from origin rebase

$git pull --rebase

Finally!

When in doubt use help

$git command help

Or visit https://training.github.com/ for official GitHub training
